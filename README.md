# Imagen a Matrices RGB en Excel
# Descripción del proyecto
Este proyecto fue desarrollado como una actividad académica en clase con el objetivo de comprender cómo las imágenes digitales pueden representarse numéricamente.
Para ello, se tomó una imagen y se procesó para obtener los valores RGB (Red, Green, Blue) de cada uno de sus píxeles. Posteriormente, estos valores se organizaron en matrices dentro de Excel, permitiendo analizar cómo se construye una imagen a partir de datos numéricos.

# Objetivo

El objetivo principal de esta actividad fue:

* Seleccionar una imagen digital.

* Convertir la imagen en información de píxeles.

* Extraer los valores RGB de cada píxel.

* Representar esos valores en matrices dentro de Excel.

* Visualizar las matrices con y sin representación de color.


# Aprendizajes

A través de esta actividad aprendimos:

* Cómo se componen las imágenes digitales a partir de píxeles.

* Cómo cada píxel está definido por valores RGB.

* Cómo representar imágenes mediante matrices numéricas.

* Cómo herramientas como Excel pueden utilizarse para visualizar estructuras de imágenes.
